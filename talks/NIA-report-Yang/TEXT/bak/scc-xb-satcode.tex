\vfill 
Experimental results show that, for most models, the saturation-based
Xie-Beerel algorithm outperforms $Lockstep$, sometimes by orders of magnitude.

Our algorithm and Lockstep improve the Xie-Beerel algorithm in different ways
\LIST{
\BULL Lockstep aims at reducing the number of image computations.
\BULL Our algorithm aims as improving state-space exploration by scheduling
event firings based on locality.
}
