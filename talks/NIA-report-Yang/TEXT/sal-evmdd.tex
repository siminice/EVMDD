\FOIL{EVMDD-based arithmetic operations}
\vfill
\LIST{
\BULL $\id{\langle MDD \rangle}$ and $\id{\langle EVMDD \rangle}$ typing:
\LIST {
\CIRC Base: \\ 
TRUE, FALSE $:= \id{\langle MDD \rangle}$ ~~~ $\mathbb{Z} := \langle EVMDD \rangle$ \\
Boolean variable $:= \id{\langle MDD \rangle}$ ~~~ Int variable $:= \langle EVMDD \rangle$ 

\CIRC MDD operations: \STRESS{$\cap, \cup, \setminus$}\\
$\id{MDDOper}:~ (\id{\langle MDD \rangle}, \cdots) \to \id{\langle MDD \rangle}$
\CIRC Arithmetic operations: \STRESS{$+,-,\times,/$} \\
$\id{ArithOper}:~(\id{\langle EVMDD \rangle}, \id{\langle EVMDD \rangle}) \to \id{\langle EVMDD \rangle}$
\CIRC Relational operations: \STRESS{$==, >, >=, <, <=$} \\
$\id{RelOper}:~(\id{\langle EVMDD \rangle}, \id{\langle EVMDD \rangle}) \to \id{\langle MDD \rangle}$
}

\vfill
\BULL Theoretical results:
\LIST{
\CIRC \STRESS{Space complexity}: For any function $f$ , the number of nodes of the EVMDD representing $f$ is at most the number
of nodes of the MTMDD representing the same function $f$.
\CIRC \STRESS{Time complexity}: The number of recursive calls of the generic apply algorithm for MTMDDs is equal to that
for EVMDDs representing the same function.
}
}
