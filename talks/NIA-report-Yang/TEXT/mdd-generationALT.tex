\emph{Locality} in events:
\LIST{
\BULL $\alpha$ is \STRESS{independent} of the $k^{th}$ submodel if:
    \LIST{
    \CIRC its enabling does not depend on $i_k$,
    \CIRC and its firing does not change the value of $i_k$.
    }
\BULL A level $k$ belongs to 
%the \STRESS{support} set of
%event $\alpha$, denoted 
$supp(\alpha)$, if $\alpha$ is not
independent of $k^{th}$ submodel.
\BULL Let $\Top(\alpha)$ be the highest-numbered level in $supp(\alpha)$.
\BULL Let  $\eset_k$ be the set of events $\{\alpha \in \eset : \Top(\alpha) = k\}$.
\BULL Let $\nset_k$ be the next-state function corresponding to all events in $\eset_k$:
$$\nset_k = \bigcup_{\alpha \in \eset_k} \nset_{\alpha}$$
}
